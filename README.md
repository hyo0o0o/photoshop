# 포토샵 기능 메모

```
Alt + 마우스휠 조절 ---> 화면크기 조절
ctrl + 0 --> 화면에 맞추기
ctrl + 1 --> 화면 100%
ctrl + j --> 레이어 복사
alt + 레이어 드래그 --> 레이어 복사
```

## 작은제목

* 메모는 이렇게 별표 치고 메모하기
- 짝대기도 가능
